# map_concat

入力されたマップの和集合を返します。複数のマップでキーが見つかった場合、この関数はこれらのマップの中で最後の値のみを保持します。これを LAST WIN と呼びます。例えば、`SELECT map_concat(map{1:3},map{1:'4'});` は `{1:"4"}` を返します。

この関数は v3.1 以降でサポートされています。

## 構文

```Haskell
ANY_MAP map_concat(any_map0, any_map1...)
```

## パラメータ

`any_mapN`: 和集合を取りたいマップの値。すべてのマップは共通の型を共有する必要があります。入力マップのデータ型が同じでない場合、戻り値の型は入力マップの共通のスーパータイプになります。

## 戻り値

入力マップの共通のスーパータイプの MAP を返します。

## 例

```Plain
mysql> SELECT map_concat(map('a',1, 'b',2), map('c',3));
+------------------------------------------+
| map_concat(map{'a':1,'b':2}, map{'c':3}) |
+------------------------------------------+
| {"c":3,"a":1,"b":2}                      |
+------------------------------------------+

mysql> select map_concat(map{1:3},map{'3.323':3});
+--------------------------------------+
| map_concat(map{1:3}, map{'3.323':3}) |
+--------------------------------------+
| {"3.323":3,"1":3}                    |
+--------------------------------------+


mysql> select map_concat(map{1:3},map{1:'4', 3:'5',null:null}, null);
+--------------------------------------------------------+
| map_concat(map{1:3}, map{1:'4',3:'5',NULL:NULL}, NULL) |
+--------------------------------------------------------+
| {null:null,1:"4",3:"5"}                                |
+--------------------------------------------------------+
```
